# Volumes

Volumes is an interactive web application designed to help you explore, track, and reflect on the Western and Global literary canon. Browse through a meticulously curated bookshelf categorized by historical periods—from Antiquity to Contemporary—and by genres. 

Features include:
- **Interactive Bookshelf**: A beautifully crafted UI for navigating through different literary eras and genres.
- **Reading Tracker**: Mark works as "Read" and keep visual track of your progress.
- **Personal Notes**: Jot down reflections, summaries, or personal thoughts for each work right within the app.
